Getting Started

Install MAGNATRIX dalam 3 langkah. Linux/macOS/Windows dengan WSL.

💻 Quick Install

# 1. Clone repository
git clone https://github.com/Magnatrix-Lab/MAGNATRIX-OS.git
cd MAGNATRIX-OS

# 2. Copy environment config dan isi API key
cp .env.example .env
nano .env # edit sesuai kebutuhan

# 3. Jalankan install script
chmod +x scripts/install.sh && ./scripts/install.sh

# 4. Boot MAGNATRIX
python3 scripts/magnatrix_boot.py

⚙ Docker Compose (Recommended)

docker-compose up -d
# 18 services akan berjalan secara paralel

💻 CLI Commands

magnatrix status # Lihat status semua layer
magnatrix swarm # Pantau swarm nodes
magnatrix trading # Status trading engine
magnatrix evolve # Trigger self-improvement cycle
magnatrix emergency # Emergency stop semua services
