# Task Brief: Untouched Domains Integration (32 modules)

## Context
MAGNATRIX-OS currently has 458 modules. We are adding 4 untouched domains with 8 modules each (32 total), bringing the system to 490 modules.

## Module Architecture Requirements
- Pure Python standard library ONLY (no external dependencies)
- Dataclass-based models
- JSON persistence via pathlib.Path
- Every module must have `to_dict()` and `get_stats()` methods
- `__all__` export list at bottom
- `*_native.py` naming convention
- Self-contained with proper error handling
- All classes must be instantiable with no arguments (default values)

## Domain 1: Code Execution Sandbox (8 modules)
Build a sandboxed code execution subsystem:

1. `sandbox_process_isolator_native.py` — Process isolation, PID namespace simulation, fork/wrap pattern
2. `sandbox_resource_limiter_native.py` — CPU time, memory, file size, open fd limits
3. `sandbox_seccomp_simulator_native.py` — System call filter simulation, allowlist/blocklist
4. `sandbox_chroot_manager_native.py` — Chroot-like directory isolation, bind mount simulation
5. `sandbox_code_runner_native.py` — Safe code execution with timeout, stdout/stderr capture
6. `sandbox_syscall_auditor_native.py` — Syscall logging, audit trail, anomaly detection
7. `sandbox_network_restrictor_native.py` — Network access restriction, DNS block, IP filter
8. `sandbox_file_system_guard_native.py` — File path allowlist, read/write restriction, temp dir enforcement

## Domain 2: OSINT Automation Framework (8 modules)
Build an open-source intelligence automation subsystem:

1. `osint_domain_recon_native.py` — Domain enumeration, subdomains, DNS records, MX/TXT parsing
2. `osint_email_harvester_native.py` — Email extraction patterns, domain-based email generation, validation
3. `osint_social_scanner_native.py` — Social media presence detection, username enumeration across platforms
4. `osint_metadata_extractor_native.py` — File metadata extraction (PDF, images), EXIF simulation, header parsing
5. `osint_dns_enumerator_native.py` — DNS brute force, zone transfer detection, record type enumeration
6. `osint_whois_parser_native.py` — WHOIS data parsing, registrar info, expiration dates, privacy detection
7. `osint_screenshot_collector_native.py` — URL list screenshot orchestration, viewport simulation, storage
8. `osint_report_generator_native.py` — OSINT report compilation, risk scoring, timeline builder, export
