# Task Brief: Untouched Domains Integration Batch 3 (16 modules)

## Context
MAGNATRIX-OS currently has 458 modules. Adding 16 more for 2 untouched domains, bringing the system toward 474+ (before Batch 1 & 2 complete).

## Module Architecture Requirements
- Pure Python standard library ONLY (no external dependencies)
- Dataclass-based models
- JSON persistence via pathlib.Path
- Every module must have `to_dict()` and `get_stats()` methods
- `__all__` export list at bottom
- `*_native.py` naming convention
- Self-contained with proper error handling
- All classes must be instantiable with no arguments (default values)

## Domain 5: Distributed Consensus (Raft-lite) (8 modules)
Build a pure-Python distributed consensus subsystem:

1. `raft_node_state_machine_native.py` — Follower/Candidate/Leader state transitions, term management, voting
2. `raft_log_replicator_native.py` — Log entry replication, append entries, log consistency check, matchIndex tracking
3. `raft_election_manager_native.py` — Randomized election timeout, vote request/response, quorum calculation
4. `raft_heartbeat_coordinator_native.py` — Leader heartbeat dispatch, timeout detection, step-down logic
5. `raft_snapshot_manager_native.py` — State machine snapshots, log compaction, snapshot installation
6. `raft_membership_changer_native.py` — Joint consensus, membership configuration changes, add/remove nodes
7. `raft_persistence_store_native.py` — Persistent state (currentTerm, votedFor, log[]), WAL simulation, recovery
8. `raft_network_simulator_native.py` — Message delay, partition simulation, packet loss, latency injection for testing

## Domain 6: Model Quantization Primitives (8 modules)
Build a pure-Python model quantization subsystem (simulated, no actual CUDA):

1. `quantization_gguf_loader_native.py` — GGUF format parser, tensor metadata, header parsing, key-value reading
2. `quantization_gptq_simulator_native.py` — GPTQ quantization simulation, weight grouping, inverse Hessian approximation
3. `quantization_awq_simulator_native.py` — AWQ activation-aware simulation, salient channel detection, scaling
4. `quantization_bit_packer_native.py` — Bit packing/unpacking, 4-bit/8-bit weight packing, LUT dequantization
5. `quantization_calibration_engine_native.py` — Calibration dataset simulation, min/max, percentile, entropy-based range
6. `quantization_layer_fuser_native.py` — Layer fusion simulation (Conv+BN, Linear+ReLU), fused kernel representation
7. `quantization_kv_cache_compressor_native.py` — KV cache quantization, per-head scaling, dynamic precision switching
8. `quantization_model_exporter_native.py` — Quantized model serialization, metadata export, compatibility check
