**Internal Architecture:**
- Event queue — heap by event time (stdlib heapq)
- Entity-Component-System (ECS) — entities are int IDs, components stored in flat arrays (array.array)
- Physics: Verlet integrator, AABB broad-phase, SAT narrow-phase
- Social: DeGroot opinion model + Watts-Strogatz small-world graph
- Economic: continuous-double-auction order book
- Snapshot: pickle ECS arrays + event heap

**Key Algorithms:**
- Verlet integration (stable for orbital + cloth-like systems)
- Barnes-Hut tree for >1000-body gravity
- DeGroot consensus
